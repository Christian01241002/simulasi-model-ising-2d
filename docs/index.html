Simulasi Model Ising 2D

Fisika Komputasi

Simulasi Model Ising 2 Dimensi menggunakan Algoritma Metropolis untuk mempelajari perubahan orientasi spin pada berbagai kondisi suhu. Proyek ini menampilkan visualisasi hasil simulasi pada fase feromagnetik, transisi fasa, dan fase paramagnetik.

Kasus 1 - Fase Feromagnetik (T = 1.0) Kasus 2 - Transisi Fasa (T = 2.27) Kasus 3 - Fase Paramagnetik (T = 4.0)

Penggunaan AI

Beberapa layanan Artificial Intelligence (AI) digunakan sebagai alat bantu dalam proses penyusunan proyek ini, antara lain ChatGPT GPT-5.5, Google Gemini, dan Claude AI. AI dimanfaatkan untuk membantu memahami konsep Model Ising 2D, memberikan referensi mengenai algoritma Metropolis, membantu penyusunan dokumentasi, memperbaiki struktur kode program, serta menyempurnakan penjelasan hasil simulasi. Seluruh hasil akhir tetap diperiksa, diuji, dan disesuaikan kembali oleh anggota kelompok.
